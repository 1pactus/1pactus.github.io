Banned Validator addresses #

Several validator addresses have been officially banned, which cannot receive rewards or perform unbond operations.

Since the tokens staked by these addresses are still included in the yield rate calculations, the actual returns will be slightly higher. These discrepancies become negligible when the total staking amount increases.
